Assignment Overview

You have been given a secret message, but how do you decrypt and read it???

Unlike regular encryption, this file has every plain letter/character you need to make sense of the study guide. Unfortunately, there are also random characters in between each good character. You will have to write code that will:

  • Read a good character from the encoded file
  • Write that good character to an output file
  • Figure out how many bad characters to skip over until you get to the next good character
  • Skip over those bad characters
  • Start the process over again until you reach the end of the file
We can always assume that the first character of the file is a good character.

How do we figure out how many characters to skip?

After uppercase letters - skip ahead by its ascii value - 61 positions
After lowercase letters - skip ahead by its ascii value - 91 positions
After numbers - skip ahead by its int() number value + 1 positions
After anything else - +1 if acsii value is even, +0 (just read the next character) if ascii value is odd

We also know that the following file helloworld.txt will decode to:

Hello world 123!!!

In a file called pa10.py write a method called decode(inputfile,outputfile).  Decode should take two parameters - both of which are strings.  The first should be the name of an encoded file (either helloworld.txt or mystery.txt or yet another file that I might use to test your code).  The second should be the name of a file that you will use as an output file.  For example:

    decode("secret.txt" , "translatedguide.txt")

Your method should read in the contents of the inputfile and, using the scheme described in the hints.txt file above, decode the hidden message, writing to the outputfile as it goes (or all at once when it is done depending on what you decide to use).

Hint: Be sure to get the helloworld.txt file to decode before working on the large studyguide.

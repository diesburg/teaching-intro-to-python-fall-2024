Program 2.3 (12 points)

Create a program named shipping.py
  • Write a Python program that calculates the shipping cost based on the weight of the package and the shipping method. Use nested if statements to determine the final shipping cost. The program should ask for two inputs: a shipping method and weight (in that order). Determine the shipping cost based on the following criteria.
  • For "standard" shipping:
    • If the weight is up to 1 kg, the cost is $5.
    • If the weight is more than 1 kg and up to 5 kg, the cost is $10.
    • If the weight is above 5 kg, the cost is $20.
  • For "express" shipping:
    • If the weight is up to 1 kg, the cost is $10.
    • If the weight is more than 1 kg and up to 5 kg, the cost is $20.
    • If the weight is above 5 kg, the cost is $40.
  • For "overnight" shipping:
    • If the weight is up to 1 kg, the cost is $25.
    • If the weight is more than 1 kg and up to 5 kg, the cost is $35.
    • If the weight is above 5 kg, the cost is $60.
  • Finally, print the cost or else the message 'That choice was not valid.'
  • Use lowercase letters "s" for standard, "e" for express, and "o" for overnight.
